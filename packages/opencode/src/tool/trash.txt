Moves a file or directory to the system Trash.

Use this for any file or directory deletion in user-facing flows. Trash is reversible; items moved to Trash can be restored from the system Trash UI.

Do not use shell deletion commands like `rm` or `del`. Those are permanent and unrecoverable, and PawWork's permission system blocks them by default.

Accepts a single file or directory path, absolute or relative to the current project directory.
